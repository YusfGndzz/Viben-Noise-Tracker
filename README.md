# Viben - Smart Noise Tracking System

Viben is a high-performance C++ application developed to maintain a productive 
work environment by monitoring real-time noise levels. It helps users or 
office managers identify peak noise times and maintain a quiet atmosphere.

### Key Features:
* **Real-time Audio Processing:** Efficient noise detection using C++.
* **Threshold Customization:** Set specific decibel limits for office standards.
* **Minimalist UI:** Clean and non-intrusive monitoring interface.
